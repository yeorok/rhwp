rhwp 테스트 4: 개발자 도구 (rhwpDev)

브라우저 개발자 도구(F12) → 콘솔 탭에서 아래 명령어를 실행하여 테스트합니다.

테스트 명령어

// 1. 도움말 출력
rhwpDev.help()

// 2. 페이지 전체 검사
rhwpDev.inspect()

// 3. 특정 링크 검사
rhwpDev.inspectLink(document.querySelector('a[data-hwp]'))

// 4. 버전 확인
rhwpDev.version

정상 링크 (개선 사항 없어야 함)

  • 완벽한 메타데이터.hwp

개선 필요 링크 (inspect()에서 경고 표시)

  • data-hwp 누락 — 확장자 폴백 감지
  • data-hwp-size 잘못된 값
  • data-hwp-pages 잘못된 값
  • data-hwp-date 잘못된 값
기대 결과: rhwpDev.inspect() 실행 시 정상 링크 1개 + 개선 사항 4건 (누락 1건, 유효성 3건)이 출력되어야 합니다.

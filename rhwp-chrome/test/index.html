rhwp Chrome 확장 — 테스트

각 테스트 페이지를 열어 Content Script가 올바르게 동작하는지 확인합니다.
테스트는 확장이 설치된 Chrome 또는 Edge에서 실행해야 합니다.

  • 01확장자 기반 자동 감지 .hwp/.hwpx 링크를 확장자만으로 감지하는지 확인
  • 02data-hwp-* 프로토콜 최소/표준/고급 적용 수준별 메타데이터 인식 확인
  • 03동적 콘텐츠 (AJAX 게시판) JavaScript로 동적 추가된 링크의 MutationObserver 감지 확인
  • 04개발자 도구 (rhwpDev) 콘솔에서 rhwpDev.inspect(), inspectLink(), help() 동작 확인
  • 05공공사이트 시뮬레이션 (정부24) 실제 공공기관 게시판과 유사한 환경에서 통합 테스트

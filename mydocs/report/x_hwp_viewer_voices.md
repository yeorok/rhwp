## 핵심 인사이트

### 1. 뷰어 접근성이 가장 큰 고통점
- 설치가 어렵고, 모바일에서 안 열리고, 유료이며, 지원이 종료됨
- HWPX 전환에도 뷰어 문제는 미해결 — 오히려 악화된 사례도
- "한컴오피스 뷰어 앱에서 hwpx를 지원 안해?"

### 2. 한컴에 대한 구조적 불신
- 독점적 지위 남용, 뷰어 유료화, 품질 저하
- 3년간(2023~2026) 해결되지 않는 문제에 대한 누적된 불만
- 기업 환경에서 라이선스 비용 부담

### 3. 시장이 이미 존재한다
- 만들어야 할 수요가 아니라, 이미 있는 수요에 답을 주는 것
- 3년간 꾸준히 동일한 불만이 반복됨 — 구조적 문제

## rhwp가 해결하는 문제

| 사용자 불만 | rhwp의 답 |
|-----------|----------|
| "뷰어 설치가 거지같다" | 설치 없음 — 브라우저에서 바로 |
| "뷰어가 유료다" | MIT 라이선스 무료 |
| "모바일에서 안 열린다" | WASM — 모든 브라우저 지원 |
| "글자 복사가 안 된다" | SVG 기반 텍스트 선택 가능 |
| "뷰어 지원이 종료됐다" | 오픈소스 — 영원히 유지 |
| "100페이지 제한" | 제한 없음 |
| "hwpx로 바꿔야 열린다" | HWP + HWPX 모두 지원 |
| "한컴 독점이 싫다" | 오픈소스로 독점 해체 |

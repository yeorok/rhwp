# Task #76: Chrome/Edge 확장 프로그램 — 2단계 완료 보고서

## 완료 항목

### 드래그 & 드롭
- rhwp-studio에 이미 구현되어 있어 추가 개발 없이 동작
- document 레벨 기본 드롭 방지 추가 — 뷰어 밖에 드롭 시 브라우저 다운로드 방지
- 확장 아이콘 → 빈 뷰어 탭 → 로컬 파일 드래그&드롭: 로딩만 수행, 다운로드 없음 ✓

### HWP 저장
- rhwp-studio의 file:save 커맨드가 이미 완전 구현됨
- File System Access API (네이티브 저장 대화상자) + Blob 다운로드 폴백
- 확장 환경에서 정상 동작 확인 ✓

### 인쇄 기능 (신규 구현)
- `renderPageSvg()` → 전체 페이지 SVG 생성 → 인쇄 전용 창에 표시
- CSS `@page` + `page-break-after: always`로 다중 페이지 인쇄
- 용지 크기 자동 감지 (문서의 페이지 정보에서 mm 단위 계산)
- 진행률 표시 ("인쇄 준비 중... (N/총페이지)")
- CSP 안전: 인라인 스크립트/이벤트 핸들러 사용 안 함 (DOM addEventListener)
- 인쇄 창 상단 바: [인쇄] [닫기] 버튼 + 파일명/페이지 수 표시
- 화면 미리보기: @media screen에서 페이지별 그림자 + 배경 표시

### Ctrl+P 단축키
- shortcut-map.ts에 `{ key: 'p', ctrl: true } → 'file:print'` 매핑 추가
- 브라우저 기본 인쇄 대화상자 대신 rhwp 인쇄 기능 실행

### 컨텍스트 메뉴 / 다운로드 가로채기
- 1단계에서 이미 구현 완료

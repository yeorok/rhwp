## 1. 배경

#139 타스크에서 수식 폰트 선정 및 적용 방안 설계가 완료되었다. 이번 타스크에서는 설계를 실제 코드에 적용한다.

현재 문제: SVG/Canvas 렌더러의 `<text>` 요소에 `font-family`가 지정되지 않아 브라우저 기본 폰트로 수식이 렌더링됨.

## 2. 목표

- `svg_render.rs`, `canvas_render.rs`에 수식 전용 font-family 적용
- Latin Modern Math woff2 번들링 및 웹 폰트 로딩
- HWP 수식 `font_name` 속성 무시 처리

## 3. font-family 체인

```
"Latin Modern Math", "STIX Two Math", "Cambria Math", "Pretendard", serif
```

## 4. 구현 단계 (3단계)

### 1단계: SVG/Canvas 렌더러에 font-family 적용
- `svg_render.rs`: 모든 `<text>` 요소에 font-family 속성 추가
- `canvas_render.rs`: `ctx.font`에 수식 폰트 반영
- 수식 전용 font-family 상수 정의

### 2단계: Latin Modern Math woff2 번들링
- Latin Modern Math OTF 다운로드 → woff2 변환
- `web/fonts/` 디렉토리에 추가
- `@font-face` 선언 추가 (local() 우선 + woff2 폴백)

### 3단계: 테스트 및 검증
- `cargo test` 전체 통과
- exam_math.hwp SVG 출력으로 폰트 적용 확인
- rhwp-studio에서 Canvas 렌더링 폰트 확인

## 5. 검증 방법

- `cargo test` 전체 통과
- SVG 출력에 font-family 속성 포함 확인
- 브라우저에서 수식 렌더링 시 Latin Modern Math 폰트 적용 확인

### Error Handling Structure

```
HwpError (native, non-WASM safe)
  ├── InvalidFile(String)
  ├── PageOutOfRange(u32)
  └── RenderError(String)

impl From<HwpError> for JsValue  ← conversion only at WASM boundary
```

- WASM functions: Return `Result<T, JsValue>` (wasm-bindgen requirement)
- Native functions (`*_native`): Return `Result<T, HwpError>` (for tests/CLI)

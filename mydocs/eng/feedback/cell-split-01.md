## Cell Splitting

After creating a 2x2 table, press F5 in cell 1,1 to select the cell.
Right-click > select Cell Split, set column count to 2 > Split.

Expected table structure:

Cell 1,1: Split into 2 cells
Cell 1,2: Unchanged
Cell 2,1: Unchanged
Cell 2,2: Unchanged

For the split cell, the width is automatically distributed as 1/2.

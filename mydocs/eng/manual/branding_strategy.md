# rhwp Branding Strategy

## Brand Identity

### Korean: 모두의 한글 (Hangul for Everyone)

**모두 (Everyone)** — People, AI, and every platform.

A project that opens the HWP format — monopolized by Hancom — so that anyone can read and write Hangul documents.

### English: rhwp

**R**ust **HWP** — The technical meaning.

The Korean pronunciation "알(R)" sounds like **"All"** — All HWP.

## Linguistic Wordplay: 알 = All

A single name carries meaning in both language spheres while pointing in the same direction.

| Language | Reading | Meaning | Connection |
|----------|---------|---------|------------|
| Korean | "알" (al) | Beginning, origin, essence | Hangul for Everyone |
| English | "R" / "All" | Rust / Everyone | All HWP |

For Korean speakers, "알" feels natural; for English speakers, "R = Rust" feels natural. And both converge on "everyone/all."

## Ancient Etymology and Symbolism of '알'

In Korean, '알' is not a mere syllable — it carries deep cultural significance.

### Birth and Beginning

- **The egg in founding myths**: Bak Hyeokgeose (Silla), King Suro (Gaya), Jumong (Goguryeo) — founders born from eggs. The egg symbolizes the start of a new world.
- **rhwp's meaning**: A project that breaks through the walls of the closed HWP format and emerges into the world — hatching from the egg.

### Knowledge and Enlightenment

- **알다 (alda)**: To know, to understand. The beginning of knowledge.
- **rhwp's meaning**: Reverse-engineering the HWP format to "figure it out" and release it as open source — turning closed knowledge into open knowledge.

### Essence and Core

- **알맹이 (almaengi)**: The core, the most essential part.
- **rhwp's meaning**: No flashy packaging — focused on the essence of the typesetting engine: accurate rendering.

### Life and Potential

- **알 (al)**: Something not yet born, but holding all possibilities within.
- **rhwp's meaning**: A project growing toward v1.0.0. It is a viewer today, but will become an editor, and eventually a document platform for everyone.

## Brand Messaging

### Tagline

**Korean**: 모두의 한글 — 알에서 시작하다 (Hangul for Everyone — Starting from the Egg)

**English**: rhwp — All HWP, Open for Everyone

### Key Messages

| Audience | Message |
|----------|---------|
| General users | View HWP files anywhere. Free, no installation required. |
| Developers | Rust + WASM-based HWP parser/renderer. MIT license. |
| Government agencies | Process HWP documents without Hancom software. |
| AI developers | An open library that lets AI read and write HWP documents. |
| Enterprises | Eliminate commercial HWP viewer license costs. |

### The Scope of "Everyone"

In "Hangul for Everyone," "everyone" means:

1. **Every person** — Freedom to open HWP files without Hancom software
2. **Every AI** — AI agents that parse and generate HWP documents directly
3. **Every platform** — Browser, VSCode, mobile, server (WASM-based)
4. **Every format** — Unified support for HWP (binary) + HWPX (XML)

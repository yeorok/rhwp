# 외부 폰트 목록

이 프로젝트에서 **폰트 메트릭스 데이터베이스 구축**에 사용하는 폰트 파일 목록이다.
저작권 보호를 위해 Git에 포함하지 않으며, 별도로 설치해야 한다.

> 웹 배포 시에는 이 폰트들을 직접 포함하지 않고, 대응되는 폰트 폴백으로 처리한다.

## ttfs/ (루트)

| 파일명 | 폰트명 | 용도 |
|--------|--------|------|
| hamchob-r.ttf | 함초롬바탕 Regular | HWP 기본 바탕체 대체 |

## ttfs/hwp/

한컴 오피스에 포함된 전용 폰트. 폰트 메트릭스(글자 폭, 높이, 자간 등) 데이터베이스 구축에 필요하다.

| 파일명 | 비고 |
|--------|------|
| HBATANG.TTF | 한컴바탕 |
| HDOTUM.TTF | 한컴돋움 |
| H2HDRM.TTF | HY헤드라인M |
| HY*.TTF | HY 시리즈 (명조, 신명조, 울릉도 등) |
| HAN*.ttf | 한양 시리즈 (백제, 솔, 소마 등) |
| ahn_*.ttf | 안상수체 |
| NanumBrush.ttf | 나눔손글씨 붓 |
| FZSong_Super.TTF | 중국어 송체 |
